Descripcion del proyecto
El proyecto es una continuacion directa del proyecto anterior de Salmontt. Esta vex resuelve la problematica de tener herramientas para registrar u consultar informacion de los empleados asociados a la empresa, pudiendo leer archivos tipo .xlsx (Excel) o .csv

Estructura general del proyecto
El proyecto esta estructurado en 3 paquetes: Model que contiene las clases direccion, persona y empleado; Data que contiene las clase gestordatoos encargado del proceso de lectura del archivo; y UI que contiene la clase main.

Instrucciones
1- Clone el repositorio desde Github https://github.com/rodrigoflo02/POO-Semana-5.git
2- Abrir el proyecto con Intellij Idea 
3- En menu, abrir el proyecto seleccionando la carpeta de este 
4- Si Desea agregar mas areas use el comando Areas.add("Nombre Area") 
5- Si desea trabajar con un excel propio asegurese de que cumpla el mismo formato en las 2 hojas e instalar el archivo en el repositorio del proyecto (Se aplica el mismo caso para archivos CSV) 
6- Ejecutar el Archivo Main.java desde el paquete UI (Ctrl + F5).
